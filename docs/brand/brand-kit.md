# The Auperating System Brand Kit

## Positioning

An AI-native operating system for autonomous projects, agents, memory, secrets, and deployment.

## Visual Direction

- Dark system core
- Electric cyan / violet signal gradients
- Glass console cards
- Kernel/grid motifs
- Premium OS-level command center aesthetic

## Palette

- Core Black: #050506
- System Graphite: #0E0D14
- Signal Cyan: #67E8F9
- Agent Violet: #745CFF
- Kernel Green: #6EE7B7
- Warning Amber: #F59E0B

## Typography

- Display: Inter Tight, Geist, or Helvetica Neue Condensed-style tracking
- Interface: Geist Sans / Inter
- Console: Geist Mono / JetBrains Mono

## Logo Concepts

1. AU monogram as an orbital command glyph.
2. A circular kernel mark with seven agent nodes around a core.
3. Wordmark: “Auperating” heavy compressed, “System” light mono.

## Image Generation Prompt

Premium futuristic AI operating system logo, black background, orbital AU monogram, cyan and violet signal glow, precise geometric grid, luxury software brand, minimal, vector-like, high contrast.
